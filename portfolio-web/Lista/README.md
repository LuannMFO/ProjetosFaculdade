# To-Do List

Um projeto simples de **lista de tarefas** desenvolvido com **HTML, CSS e JavaScript**.  
Permite adicionar, marcar como concluída e excluir tarefas diretamente no navegador.

---

## Funcionalidades

- Adicionar tarefas com o botão **Adicionar** ou pressionando **Enter**.  
- Marcar tarefas como **concluídas** ao clicar nelas.  
- Excluir tarefas individualmente.  
- Layout limpo, moderno e responsivo.

---

## Tecnologias Utilizadas

- **HTML5** – estrutura da página.  
- **CSS3** – estilização e layout responsivo.  
- **JavaScript** – lógica da aplicação e manipulação do DOM.
